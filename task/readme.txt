For executing the icoaSpeaker in the following platforms:

windows:
we need to download the following files:
pywin32-219.win32-py2.7
pyttsx-1.1-py2-none-any.whl

run the following downloaded exe file in commandline : pywin32-219.win32-py2.7
install pyttsx from the file downloaded : 
pyttsx-1.1-py2-none-any.whl, by the following syntax in commandline:
pip install filePath\pyttsx-1.1-py2-none-any.whl

linux:

install the module by the code in terminal:
pip install pyttsx
